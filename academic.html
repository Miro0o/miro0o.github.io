About

I am an MSc student in Computer Science and Engineering at DTU. My interests sit at the intersection of software and systems security, program analysis, and LLM-based developer tools. I am especially interested in using language models to make security analysis and testing more adaptable across programming languages.

Before DTU, I studied cybersecurity at Sichuan University and spent a semester as a visiting student at UC Berkeley. I have worked on eBPF monitoring, cloud and network performance, penetration testing, static malware analysis, and multi-language fuzzing.

Selected projects

Large Language Model Based Multi-Language Fuzzing

2024–2025

Bachelor's thesis · Supervisor: Prof. Yi Zhang

A universal fuzzing framework that uses prompt engineering to generate diverse test cases for systems implemented in different languages, with the goal of improving code coverage and language support over conventional fuzzing workflows.

LLM Agents for Static Malware Analysis

2024

Course project · Supervisor: Prof. Zhiyang Fang

A static-analysis workflow using agentic language models to support malware detection for ArkTS and HarmonyOS applications, combining signature-based analysis with semantic reasoning over code.

DPDK Test Suites

2024

Intern project · NSFOCUS systems group

Performance benchmarking tools for DPDK network drivers and applications, including automated packet-processing measurements and latency reporting.

OpenFlow SDN Controller for Security Policy Management

2022–2023

Course project · code

An OpenFlow 1.0 controller built with RYU and Mininet, using EAPoL, access-control lists, and role-based access control to simplify network security policy management.
